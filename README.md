# Homebridge Platform AirPatrol Plugin
This is a plugin for AirPatrol wifi heater/cooler controller

# Support for
0.1.0
* Starting

# Default config
```json
"platforms": [
    {
        "name": "AirPatrol Plugin",
        "Token": "[Token]",
        "UpdateTime": 60,
        "Debug": false,
        "platform": "AirPatrol"
    }
]
```
